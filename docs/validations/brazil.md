# Brazil Validations

**Type:** `String`  
**Interface:** `BrazilValidations`

| Method | Description |
|--------|-------------|
| `cpf(message)` | Valid CPF |
| `cnpj(message)` | Valid CNPJ |
| `cep(message)` | Valid CEP (8 digits; accepts mask, e.g. `12345-678`) |
| `phoneBr(message)` | Valid Brazilian phone (10 or 11 digits, valid DDD) |
