# Boolean Validations

**Type:** `Boolean`  
**Interface:** `BooleanValidations`

| Method | Description |
|--------|-------------|
| `isTrue(message)` | Value must be `Boolean.TRUE` |
| `isFalse(message)` | Value must be `Boolean.FALSE` |

> `null` fails both validations.
